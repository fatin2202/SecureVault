# Secure Vault

## Project Description
Secure Vault is a Python-based application used to securely store passwords and sensitive information.

## Features
- Secure password storage
- Simple user interface
- Fast access to saved passwords
- Windows executable (.exe) available

## Technologies Used
- Python
- Tkinter
- PyInstaller

## How to Run
1. Open the `dist` folder.
2. Double-click `secure-vault.exe`.
3. Use the application to store and manage passwords securely.
